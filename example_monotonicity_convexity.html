例题1：函数单调性判定

确定函数f(x) = 2x³ - 9x² + 12x - 3的单调区间

解答：

1. 求函数的定义域：x ∈ (-∞, +∞)

2. 求一阶导数：
f'(x) = 6x² - 18x + 12 = 6(x² - 3x + 2) = 6(x - 1)(x - 2)

3. 求临界点：令f'(x) = 0，解得x = 1和x = 2

4. 划分区间并判定符号：
- 当x < 1时，(x - 1) < 0，(x - 2) < 0，所以f'(x) > 0，函数在(-∞, 1)上单调递增
- 当1 < x < 2时，(x - 1) > 0，(x - 2) < 0，所以f'(x) < 0，函数在(1, 2)上单调递减
- 当x > 2时，(x - 1) > 0，(x - 2) > 0，所以f'(x) > 0，函数在(2, +∞)上单调递增

因此，函数的单调递增区间为(-∞, 1)和(2, +∞)，单调递减区间为(1, 2)

例题2：函数极值求解

求函数f(x) = x⁴ - 2x² + 3的极值

解答：

1. 求函数的定义域：x ∈ (-∞, +∞)

2. 求一阶导数：
f'(x) = 4x³ - 4x = 4x(x² - 1) = 4x(x - 1)(x + 1)

3. 求临界点：令f'(x) = 0，解得x = -1, x = 0, x = 1

4. 求二阶导数：
f''(x) = 12x² - 4

5. 利用二阶导数判定极值：
- 在x = -1处，f''(-1) = 12(-1)² - 4 = 12 - 4 = 8 > 0，所以x = -1是极小值点
极小值f(-1) = (-1)⁴ - 2(-1)² + 3 = 1 - 2 + 3 = 2

- 在x = 0处，f''(0) = 12(0)² - 4 = -4 < 0，所以x = 0是极大值点
极大值f(0) = 0⁴ - 2(0)² + 3 = 3

- 在x = 1处，f''(1) = 12(1)² - 4 = 12 - 4 = 8 > 0，所以x = 1是极小值点
极小值f(1) = 1⁴ - 2(1)² + 3 = 1 - 2 + 3 = 2

因此，函数的极大值为3（在x=0处取得），极小值为2（在x=-1和x=1处取得）

例题3：函数凸凹性判定

确定函数f(x) = x³ - 3x² - 9x + 5的凸凹区间和拐点

解答：

1. 求函数的定义域：x ∈ (-∞, +∞)

2. 求一阶导数：
f'(x) = 3x² - 6x - 9 = 3(x² - 2x - 3) = 3(x - 3)(x + 1)

3. 求二阶导数：
f''(x) = 6x - 6 = 6(x - 1)

4. 求二阶导数为零的点：令f''(x) = 0，解得x = 1

5. 划分区间并判定凸凹性：
- 当x < 1时，f''(x) < 0，函数在(-∞, 1)上是凹函数
- 当x > 1时，f''(x) > 0，函数在(1, +∞)上是凸函数

6. 确定拐点：
在x = 1处，二阶导数由负变正，因此点(1, f(1))是拐点
计算f(1) = 1³ - 3(1)² - 9(1) + 5 = 1 - 3 - 9 + 5 = -6
所以拐点为(1, -6)

因此，函数的凹区间为(-∞, 1)，凸区间为(1, +∞)，拐点为(1, -6)

例题4：函数图像描绘

描绘函数f(x) = x/(1 + x²)的图像

解答：

1. 确定定义域：x ∈ (-∞, +∞)

2. 检查对称性：
f(-x) = (-x)/(1 + (-x)²) = -x/(1 + x²) = -f(x)，所以函数是奇函数，图像关于原点对称

3. 求截距：
- 与x轴交点：令f(x) = 0，解得x = 0
- 与y轴交点：f(0) = 0，所以截距为(0, 0)

4. 求渐近线：
- 水平渐近线：limₓ→±∞ x/(1 + x²) = 0，所以y = 0是水平渐近线
- 垂直渐近线：分母1 + x²永远不为零，所以没有垂直渐近线

5. 求一阶导数和单调区间：
f'(x) = [1·(1 + x²) - x·2x]/(1 + x²)² = (1 - x²)/(1 + x²)² = (1 - x)(1 + x)/(1 + x²)²
临界点：x = -1, x = 1
单调性：
- 当x < -1或x > 1时，f'(x) < 0，函数单调递减
- 当-1 < x < 1时，f'(x) > 0，函数单调递增
极值：
- x = -1时，f(-1) = -1/2，极小值
- x = 1时，f(1) = 1/2，极大值

6. 求二阶导数和凸凹区间：
f''(x) = [(-2x)(1 + x²)² - (1 - x²)·2(1 + x²)·2x]/(1 + x²)⁴
化简得：f''(x) = 2x(x² - 3)/(1 + x²)³
二阶导数为零的点：x = 0, x = -√3, x = √3
凸凹性：
- 当x < -√3或0 < x < √3时，f''(x) < 0，函数是凹函数
- 当-√3 < x < 0或x > √3时，f''(x) > 0，函数是凸函数
拐点：(0, 0), (-√3, -√3/4), (√3, √3/4)

7. 综合以上信息绘制图像：
- 关于原点对称
- 过原点(0, 0)
- 以y = 0为水平渐近线
- 在(-∞, -1)和(1, +∞)上单调递减，在(-1, 1)上单调递增
- 在(-∞, -√3)和(0, √3)上是凹函数，在(-√3, 0)和(√3, +∞)上是凸函数
- 极小值点(-1, -1/2)，极大值点(1, 1/2)
- 拐点(0, 0), (-√3, -√3/4), (√3, √3/4)

例题5：利用单调性证明不等式

证明：当x > 0时，ln(1 + x) < x - x²/2 + x³/3

解答：

1. 构造辅助函数：
f(x) = x - x²/2 + x³/3 - ln(1 + x)

2. 求一阶导数：
f'(x) = 1 - x + x² - 1/(1 + x)
= [(1 - x + x²)(1 + x) - 1]/(1 + x)
= [1 + x - x - x² + x² + x³ - 1]/(1 + x)
= x³/(1 + x)

3. 分析一阶导数的符号：
当x > 0时，x³ > 0，1 + x > 0，所以f'(x) > 0
因此，函数f(x)在(0, +∞)上单调递增

4. 计算f(0)：
f(0) = 0 - 0 + 0 - ln(1 + 0) = 0 - 0 = 0

5. 由于f(x)在(0, +∞)上单调递增，且f(0) = 0，所以当x > 0时，f(x) > 0
即：x - x²/2 + x³/3 - ln(1 + x) > 0
移项得：ln(1 + x) < x - x²/2 + x³/3

因此，当x > 0时，ln(1 + x) < x - x²/2 + x³/3成立
