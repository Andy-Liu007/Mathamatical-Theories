例题1：微分运算法则的应用

求下列函数的微分：

  1. y = x³ + 2x² - 5x + 1
  2. y = x²sinx
  3. y = eˣ / x

解答：

1. 对于y = x³ + 2x² - 5x + 1，使用微分的线性法则：d(u ± v) = du ± dv
dy = d(x³) + d(2x²) - d(5x) + d(1) = 3x²dx + 4xdx - 5dx + 0 = (3x² + 4x - 5)dx

2. 对于y = x²sinx，使用微分的乘积法则：d(uv) = u dv + v du
令u = x²，v = sinx，则du = 2xdx，dv = cosxdx
dy = x² cosxdx + sinx · 2xdx = x(xcosx + 2sinx)dx

3. 对于y = eˣ / x，使用微分的商法则：d(u/v) = (v du - u dv) / v²
令u = eˣ，v = x，则du = eˣdx，dv = dx
dy = (x eˣdx - eˣ dx) / x² = eˣ(x - 1)dx / x²

例题2：一阶微分的形式不变性

已知y = sin(u)，u = x² + 1，分别用两种方法求dy。

解答：

方法一：先复合再求微分
y = sin(x² + 1)
dy = cos(x² + 1) · d(x² + 1) = cos(x² + 1) · 2xdx = 2x cos(x² + 1)dx

方法二：利用一阶微分的形式不变性
首先对y = sin(u)求微分：dy = cos(u) du
然后对u = x² + 1求微分：du = 2xdx
将du代入dy的表达式：dy = cos(u) · 2xdx = cos(x² + 1) · 2xdx = 2x cos(x² + 1)dx

两种方法得到的结果一致，验证了一阶微分的形式不变性。

例题3：隐函数求导法

求由方程x² + y² = r²（r为常数）确定的隐函数y = y(x)的导数dy/dx。

解答：

对方程两边同时微分，利用微分的运算法则：
d(x² + y²) = d(r²)
d(x²) + d(y²) = 0

计算各项微分：
2xdx + 2ydy = 0

解出dy/dx：
2ydy = -2xdx
dy = (-2x / 2y)dx = (-x / y)dx

因此，dy/dx = -x / y

例题4：微分的应用 - 近似计算

利用微分近似计算sin(31°)的值，已知sin(30°) = 0.5，cos(30°) = √3/2 ≈ 0.8660。

解答：

设f(x) = sinx，取x₀ = 30° = π/6 rad，Δx = 1° = π/180 rad ≈ 0.01745 rad

首先求f(x)的导数：f'(x) = cosx

计算f(x₀)和f'(x₀)：
f(π/6) = sin(π/6) = 0.5
f'(π/6) = cos(π/6) = √3/2 ≈ 0.8660

根据微分近似公式：f(x₀+Δx) ≈ f(x₀) + f'(x₀)Δx

代入数值：
sin(31°) = sin(π/6 + π/180) ≈ sin(π/6) + cos(π/6)·(π/180)
≈ 0.5 + 0.8660·0.01745 ≈ 0.5 + 0.01511 ≈ 0.5151

因此，sin(31°) ≈ 0.5151（精确值约为0.51503807491）

例题5：微分的应用 - 误差估计

用卡尺测量球体的直径d = 10cm，测量误差为Δd = ±0.1cm，求球体体积的误差。

解答：

球体的体积公式为：V = (4/3)πr³ = (4/3)π(d/2)³ = (1/6)πd³

首先求体积V关于直径d的导数：dV/dd = (1/6)π·3d² = (1/2)πd²

根据微分的定义，体积的误差ΔV ≈ dV = (dV/dd)Δd = (1/2)πd²Δd

代入数值：d = 10cm，Δd = ±0.1cm
ΔV ≈ (1/2)π·10²·(±0.1) = (1/2)π·100·(±0.1) = 5π·(±0.1) = ±0.5π ≈ ±1.57 cm³

因此，球体体积的误差约为±1.57 cm³
