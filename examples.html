定理1：函数极限的唯一性

例题1：证明极限的唯一性
设limₓ→0 f(x) = 0，且limₓ→0 f(x) = 1，证明这是不可能的。
由函数极限的唯一性定理，若limₓ→a f(x) = A 且 limₓ→a f(x) = B，则A = B
这里假设limₓ→0 f(x) = 0 且 limₓ→0 f(x) = 1，显然0 ≠ 1，与唯一性定理矛盾
因此假设不成立，这是不可能的
例题2：唯一性定理的应用
已知limₓ→1 f(x) = 2，且f(x) = (x² - 1)/(x - 1) + c，求c的值。
首先化简f(x)：(x² - 1)/(x - 1) = (x - 1)(x + 1)/(x - 1) = x + 1（x ≠ 1）
因此limₓ→1 f(x) = limₓ→1 (x + 1 + c) = 1 + 1 + c = 2 + c
已知limₓ→1 f(x) = 2，由唯一性定理，2 + c = 2 ⇒ c = 0
