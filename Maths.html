极限与连续

▶ 数列极限的定义
定义
∀ε > 0, ∃N ∈ ℕ, 使得当 n > N 时，|aₙ - A| < ε
对于任意给定的正数ε，总存在正整数N，使得当n大于N时，数列{aₙ}的项aₙ与常数A的距离小于ε，则称数列{aₙ}收敛于A，记为limₙ→∞ aₙ = A
▶ 证明概要
证明思路
1. 给定任意ε > 0
2. 寻找合适的N，使得当n > N时，|aₙ - A| < ε
3. 通常通过解不等式|aₙ - A| < ε来确定N的取值
4. 验证所选N满足条件

收敛数列的极限是唯一的

假设A ≠ B，取ε = |A - B|/2 > 0
由limₙ→∞ aₙ = A，∃N₁ ∈ ℕ，当n > N₁时，|aₙ - A| < ε
由limₙ→∞ aₙ = B，∃N₂ ∈ ℕ，当n > N₂时，|aₙ - B| < ε
取N = max{N₁, N₂}，则当n > N时：
|A - B| ≤ |A - aₙ| + |aₙ - B| < ε + ε = |A - B|，矛盾
故A = B

若limₙ→∞ aₙ = A，则∃M > 0，使得∀n ∈ ℕ，|aₙ| ≤ M
收敛数列必有界

▶ 主要定理3：Cauchy收敛准则
定理表述
数列{aₙ}收敛 ⇨ ∀ε > 0, ∃N ∈ ℕ, 使得当m, n > N时，|aₘ - aₙ| < ε
数列收敛的充要条件是：对于任意给定的正数ε，总存在正整数N，使得当m, n都大于N时，aₘ与aₙ的距离小于ε

▶ 主要定理4：Bolzano-Weierstrass定理
定理表述
任何有界数列必有收敛子列
这是实数系的基本定理之一，保证了有界数列中存在收敛的子列
▶ 主要推论3：收敛数列的子列收敛性
定理表述
若limₙ→∞ aₙ = A，则{aₙ}的任意子列{aₙₖ}也收敛于A
收敛数列的所有子列都收敛到同一个极限
▶ 相关引理
▶ 引理1：极限的保号性
定理表述
若limₙ→∞ aₙ = A > 0（或A < 0），则∃N ∈ ℕ，当n > N时，aₙ > 0（或aₙ < 0）
▶ 引理2：夹逼准则
定理表述
若∃N ∈ ℕ，当n > N时，bₙ ≤ aₙ ≤ cₙ，且limₙ→∞ bₙ = limₙ→∞ cₙ = A，则limₙ→∞ aₙ = A
▶ 引理3：极限的保序性
定理表述
若limₙ→∞ aₙ = A，limₙ→∞ bₙ = B，且A < B，则∃N ∈ ℕ，当n > N时，aₙ < bₙ
▶ 引理4：单调子列定理
定理表述
任何数列都存在单调子列
这是证明Bolzano-Weierstrass定理的关键引理
▶ 数列极限的扩展应用
应用领域
数列极限在数学分析、微积分、物理学等领域有广泛应用
▶ 应用1：无穷级数的收敛性
定义
无穷级数是数列各项之和的极限形式，它与数列极限有着密切的联系
部分和数列
Sₙ = a₁ + a₂ + a₃ + ... + aₙ = ∑ᵢ=1ⁿ aᵢ
对于无穷级数∑ₙ=1^∞ aₙ，我们定义其前n项和为Sₙ，称数列{Sₙ}为该级数的部分和数列
收敛性定义
无穷级数∑ₙ=1^∞ aₙ收敛 ⇨ 存在常数S，使得limₙ→∞ Sₙ = S
如果无穷级数的部分和数列{Sₙ}收敛于某个常数S，那么我们称该无穷级数收敛，且其和为S；否则称其发散
示例
几何级数∑ₙ=0^∞ arⁿ，当|r| < 1时收敛，和为a/(1-r)；当|r| ≥ 1时发散
例如：∑ₙ=0^∞ (1/2)ⁿ = 1 + 1/2 + 1/4 + 1/8 + ... = 2
收敛的必要条件
若∑ₙ=1^∞ aₙ收敛，则limₙ→∞ aₙ = 0
注意：这只是必要条件，而非充分条件。例如调和级数∑ₙ=1^∞ 1/n，虽然limₙ→∞ 1/n = 0，但级数发散
▶ 函数的极限
定义
∀ε > 0, ∃δ > 0, 使得当 0 < |x - a| < δ 时，|f(x) - A| < ε
对于任意给定的正数ε，总存在正数δ，使得当x与a的距离小于δ且x≠a时，f(x)与A的距离小于ε，则称函数f(x)当x趋近于a时以A为极限，记为limₓ→a f(x) = A
▶ 主要定理1：函数极限的唯一性
定理表述
若limₓ→a f(x) = A 且 limₓ→a f(x) = B，则A = B
函数在某点的极限是唯一的
证明
假设A ≠ B，取ε = |A - B|/2 > 0
由limₓ→a f(x) = A，∃δ₁ > 0，当0 < |x - a| < δ₁时，|f(x) - A| < ε
由limₓ→a f(x) = B，∃δ₂ > 0，当0 < |x - a| < δ₂时，|f(x) - B| < ε
取δ = min{δ₁, δ₂}，则当0 < |x - a| < δ时：
|A - B| = |(A - f(x)) + (f(x) - B)| ≤ |A - f(x)| + |f(x) - B| < ε + ε = |A - B|，矛盾
故A = B

▶ 主要定理2：函数极限的局部有界性
定理表述
若limₓ→a f(x) = A，则∃δ > 0 和 M > 0，使得当 0 < |x - a| < δ 时，|f(x)| ≤ M
函数在极限点附近是有界的
证明
取ε = 1，由limₓ→a f(x) = A，∃δ > 0，当0 < |x - a| < δ时，|f(x) - A| < 1
由三角不等式得：|f(x)| = |(f(x) - A) + A| ≤ |f(x) - A| + |A| < 1 + |A|
令M = 1 + |A|，则当0 < |x - a| < δ时，|f(x)| ≤ M
故f(x)在a的δ邻域内（去心）有界

▶ 主要定理3：函数极限的局部保号性
定理表述
若limₓ→a f(x) = A > 0（或A < 0），则∃δ > 0，使得当 0 < |x - a| < δ 时，f(x) > 0（或f(x) < 0）
函数在极限点附近保持与极限值相同的符号
证明
仅证A > 0的情况（A < 0类似可证）
取ε = A/2 > 0，由limₓ→a f(x) = A，∃δ > 0，当0 < |x - a| < δ时，|f(x) - A| < ε
则f(x) > A - ε = A - A/2 = A/2 > 0
故当0 < |x - a| < δ时，f(x) > 0

▶ 主要定理4：函数极限的四则运算法则
定理表述
若limₓ→a f(x) = A，limₓ→a g(x) = B，则：
1. limₓ→a [f(x) ± g(x)] = A ± B
2. limₓ→a [f(x)·g(x)] = A·B
3. 若B ≠ 0，则limₓ→a [f(x)/g(x)] = A/B
证明
1. 证明和的极限：limₓ→a [f(x) + g(x)] = A + B
∀ε > 0，由limₓ→a f(x) = A，∃δ₁ > 0，当0 < |x - a| < δ₁时，|f(x) - A| < ε/2
由limₓ→a g(x) = B，∃δ₂ > 0，当0 < |x - a| < δ₂时，|g(x) - B| < ε/2
取δ = min{δ₁, δ₂}，则当0 < |x - a| < δ时：
|(f(x) + g(x)) - (A + B)| ≤ |f(x) - A| + |g(x) - B| < ε/2 + ε/2 = ε
故limₓ→a [f(x) + g(x)] = A + B

2. 证明积的极限：limₓ→a [f(x)·g(x)] = A·B
由局部有界性，∃δ₀ > 0，M > 0，当0 < |x - a| < δ₀时，|g(x)| ≤ M
∀ε > 0，取δ₁ > 0，当0 < |x - a| < δ₁时，|f(x) - A| < ε/(2(M + 1))
取δ₂ > 0，当0 < |x - a| < δ₂时，|g(x) - B| < ε/(2|A| + 1))
取δ = min{δ₀, δ₁, δ₂}，则当0 < |x - a| < δ时：
|f(x)g(x) - AB| = |(f(x) - A)g(x) + A(g(x) - B)| ≤ |f(x) - A||g(x)| + |A||g(x) - B| < ε/2 + ε/2 = ε
故limₓ→a [f(x)·g(x)] = A·B

3. 证明商的极限：limₓ→a [f(x)/g(x)] = A/B (B ≠ 0)
由局部保号性，∃δ₀ > 0，当0 < |x - a| < δ₀时，|g(x)| > |B|/2
令h(x) = 1/g(x)，则|h(x) - 1/B| = |B - g(x)|/(|g(x)||B|) < 2|B - g(x)|/B²
由limₓ→a g(x) = B，∀ε > 0，∃δ₁ > 0，当0 < |x - a| < δ₁时，|g(x) - B| < εB²/2
取δ = min{δ₀, δ₁}，则当0 < |x - a| < δ时，|h(x) - 1/B| < ε
故limₓ→a 1/g(x) = 1/B，再由积的法则得：
limₓ→a [f(x)/g(x)] = limₓ→a [f(x)·(1/g(x))] = A·(1/B) = A/B
▶ 主要定理5：函数极限的夹逼准则
定理表述
若∃δ > 0，当 0 < |x - a| < δ 时，g(x) ≤ f(x) ≤ h(x)，且limₓ→a g(x) = limₓ→a h(x) = A，则limₓ→a f(x) = A
证明
∀ε > 0，由limₓ→a g(x) = A，∃δ₁ > 0，当0 < |x - a| < δ₁时，|g(x) - A| < ε ⇒ A - ε < g(x)
由limₓ→a h(x) = A，∃δ₂ > 0，当0 < |x - a| < δ₂时，|h(x) - A| < ε ⇒ h(x) < A + ε
设δ₀是定理中给出的正数，取δ = min{δ₀, δ₁, δ₂}，则当0 < |x - a| < δ时：
A - ε < g(x) ≤ f(x) ≤ h(x) < A + ε ⇒ |f(x) - A| < ε
故limₓ→a f(x) = A
▶ 主要定理6：Heine定理（数列与函数极限的关系）
定理表述
limₓ→a f(x) = A ⇨ 对任意数列{xₙ}→a（xₙ≠a），有limₙ→∞ f(xₙ) = A
函数极限与数列极限的桥梁，将函数极限问题转化为数列极限问题
证明
必要性（⇒）：
已知limₓ→a f(x) = A，即∀ε > 0，∃δ > 0，当0 < |x - a| < δ时，|f(x) - A| < ε
对任意数列{xₙ}→a（xₙ≠a），由数列极限定义，∃N ∈ ℕ，当n > N时，0 < |xₙ - a| < δ
因此，当n > N时，|f(xₙ) - A| < ε，故limₙ→∞ f(xₙ) = A

充分性（⇐）：
反证法，假设limₓ→a f(x) ≠ A，则∃ε₀ > 0，∀δ > 0，∃x满足0 < |x - a| < δ，但|f(x) - A| ≥ ε₀
取δₙ = 1/n，n = 1,2,3,...，则∃xₙ满足0 < |xₙ - a| < 1/n，但|f(xₙ) - A| ≥ ε₀
此时数列{xₙ}→a（xₙ≠a），但limₙ→∞ f(xₙ) ≠ A，与条件矛盾
故limₓ→a f(x) = A
▶ 相关推论
▶ 推论1：函数极限的保序性
定理表述
若limₓ→a f(x) = A，limₓ→a g(x) = B，且A < B，则∃δ > 0，当 0 < |x - a| < δ 时，f(x) < g(x)
▶ 推论2：复合函数的极限
定理表述
若limᵤ→u₀ f(u) = A，limₓ→x₀ g(x) = u₀，且∃δ > 0，当 0 < |x - x₀| < δ 时，g(x) ≠ u₀，则limₓ→x₀ f(g(x)) = A
▶ 连续函数
连续的定义
函数f(x)在点a连续 ⇨ limₓ→a f(x) = f(a)
函数在某点连续当且仅当函数在该点的极限等于函数在该点的函数值
初等函数的连续性
基本初等函数（幂函数、指数函数、对数函数、三角函数、反三角函数）在其定义域内都是连续的
初等函数 = 基本初等函数经过有限次四则运算和复合运算所得到的函数
初等函数在其定义域的区间内都是连续的
连续函数的性质
1. 连续函数的和、差、积、商（分母不为零）仍是连续函数
2. 连续函数的复合函数仍是连续函数
3. 闭区间上的连续函数有最值定理和介值定理
▶ 间断点
间断点的定义
如果函数f(x)在点a不连续，则称a为f(x)的间断点
间断点的分类
第一类间断点：左极限和右极限都存在
1. 可去间断点：limₓ→a⁻ f(x) = limₓ→a⁺ f(x) ≠ f(a) 或 f(a)无定义
2. 跳跃间断点：limₓ→a⁻ f(x) ≠ limₓ→a⁺ f(x)
第二类间断点：左极限或右极限至少有一个不存在
1. 无穷间断点：limₓ→a f(x) = ∞ 或 limₓ→a f(x) = -∞
2. 振荡间断点：limₓ→a f(x)不存在且不为无穷大，如sin(1/x)在x=0处
▶ 无穷小量与无穷大量
无穷小量的定义
limₓ→a α(x) = 0 ⇨ α(x)是x→a时的无穷小量
无穷大量的定义
limₓ→a f(x) = ∞ ⇨ f(x)是x→a时的无穷大量
无穷大量与无穷小量的关系：非零无穷小量的倒数是无穷大量，无穷大量的倒数是无穷小量
无穷小量的阶
设α(x)和β(x)都是x→a时的无穷小量，且β(x) ≠ 0
1. 高阶无穷小：limₓ→a α(x)/β(x) = 0 ⇨ α(x) = o(β(x))(x→a)
2. 低阶无穷小：limₓ→a α(x)/β(x) = ∞ ⇨ α(x)是β(x)的低阶无穷小
3. 同阶无穷小：limₓ→a α(x)/β(x) = C ≠ 0 ⇨ α(x) ~ Cβ(x)(x→a)
4. 等价无穷小：limₓ→a α(x)/β(x) = 1 ⇨ α(x) ~ β(x)(x→a)
常见等价无穷小（x→0）
sinx ~ x, tanx ~ x, arcsinx ~ x, arctanx ~ x
ln(1+x) ~ x, eˣ - 1 ~ x, 1 - cosx ~ x²/2
(1+x)ᵃ - 1 ~ ax
▶ 曲线的渐近线
水平渐近线
若limₓ→+∞ f(x) = b 或 limₓ→-∞ f(x) = b，则y = b是曲线y = f(x)的水平渐近线
垂直渐近线
若limₓ→a⁺ f(x) = ±∞ 或 limₓ→a⁻ f(x) = ±∞，则x = a是曲线y = f(x)的垂直渐近线
斜渐近线
若limₓ→+∞ [f(x)/x] = k 且 limₓ→+∞ [f(x) - kx] = b，则y = kx + b是曲线y = f(x)的斜渐近线
或limₓ→-∞ [f(x)/x] = k 且 limₓ→-∞ [f(x) - kx] = b，则y = kx + b是曲线y = f(x)的斜渐近线
示例
求曲线y = (x² + 1)/(x - 1)的渐近线
1. 垂直渐近线：x = 1（分母为零，limₓ→1 f(x) = ∞）
2. 斜渐近线：k = limₓ→∞ (x²+1)/(x(x-1)) = 1，b = limₓ→∞ [(x²+1)/(x-1) - x] = 1，故y = x + 1
3. 无水平渐近线（limₓ→∞ f(x) = ∞）

微分与导数

▶ 导数与微分的定义
导数的定义
f'(x₀) = limₕ→₀ [f(x₀+h) - f(x₀)] / h
函数f(x)在x₀处的导数表示函数在该点的瞬时变化率
▶ 可导与连续的关系
定理表述
若函数f(x)在x₀处可导，则f(x)在x₀处连续
可导是连续的充分条件，但连续不是可导的充分条件
▶ 微分的定义
定义
df(x) = f'(x)dx
微分表示函数在某点处的线性近似增量
▶ 微分的几何意义
几何解释
微分的几何意义是曲线在某点处切线的纵坐标增量。当Δx很小时，曲线y=f(x)在点(x₀,f(x₀))处的函数增量Δy可以用切线的纵坐标增量dy近似表示。
Δy = f(x₀+Δx) - f(x₀) ≈ dy = f'(x₀)Δx
在几何上，dy是切线在x₀到x₀+Δx之间的纵坐标变化，而Δy是曲线在该区间内的纵坐标变化。当Δx趋近于0时，两者的差是Δx的高阶无穷小。

▶ 导数的运算
基本导数公式
(C)' = 0, (xⁿ)' = nxⁿ⁻¹, (sinx)' = cosx, (cosx)' = -sinx
(eˣ)' = eˣ, (lnx)' = 1/x, (aˣ)' = aˣlna, (logₐx)' = 1/(x lna)
▶ 导数的四则运算
运算法则
[u(x) ± v(x)]' = u'(x) ± v'(x)
[u(x)v(x)]' = u'(x)v(x) + u(x)v'(x)
[u(x)/v(x)]' = [u'(x)v(x) - u(x)v'(x)] / v²(x) (v(x) ≠ 0)
▶ 复合函数求导法则
链式法则
若y = f(u), u = g(x)，则dy/dx = dy/du · du/dx
▶ 反函数的求导法则
定理表述
若函数y = f(x)在区间I内单调、可导且f'(x) ≠ 0，则其反函数x = f⁻¹(y)在对应区间J内也可导，且：
(f⁻¹)'(y) = 1 / f'(x)
反函数的导数等于原函数导数的倒数
示例
已知y = sinx，求其反函数x = arcsiny的导数
dy/dx = cosx，故dx/dy = 1/cosx = 1/√(1 - sin²x) = 1/√(1 - y²)
▶ 对数求导法
方法介绍
对数求导法是一种用于求复杂函数导数的方法，特别是对于幂指函数、多个函数的乘积或商的形式。其基本思想是先取对数，利用对数的性质简化函数，再求导。
步骤
1. 对函数y = f(x)两边取自然对数：ln|y| = ln|f(x)|
2. 利用对数性质化简右边：ln|y| = g(x)
3. 两边对x求导：(1/y)·y' = g'(x)
4. 解出y'：y' = y·g'(x) = f(x)·g'(x)
示例
求y = xˣ的导数
1. 取对数：ln|y| = x·lnx
2. 两边求导：(1/y)·y' = lnx + 1
3. 解出y'：y' = y·(lnx + 1) = xˣ·(lnx + 1)

▶ 高阶导数
定义
函数的高阶导数是指对函数多次求导得到的结果。如果函数f(x)的导数f'(x)仍然可导，则称f'(x)的导数为f(x)的二阶导数，记为f''(x)或d²y/dx²。以此类推，可以定义三阶导数、四阶导数等。
f''(x) = limₕ→₀ [f'(x+h) - f'(x)] / h
f⁽ⁿ⁾(x) = limₕ→₀ [f⁽ⁿ⁻¹⁾(x+h) - f⁽ⁿ⁻¹⁾(x)] / h
▶ 常见函数的高阶导数
1. 幂函数：(xⁿ)⁽ᵏ⁾ = n(n-1)...(n-k+1)xⁿ⁻ᵏ (k ≤ n), 若k > n则为0
2. 指数函数：(eˣ)⁽ⁿ⁾ = eˣ
3. 正弦函数：(sinx)⁽ⁿ⁾ = sin(x + nπ/2)
4. 余弦函数：(cosx)⁽ⁿ⁾ = cos(x + nπ/2)
5. 对数函数：(lnx)⁽ⁿ⁾ = (-1)ⁿ⁻¹(n-1)! / xⁿ
▶ Leibniz公式（乘积的n阶导数）
公式
(uv)⁽ⁿ⁾ = Σₖ=₀ⁿ Cₙᵏ u⁽ⁿ⁻ᵏ⁾ v⁽ᵏ⁾
其中Cₙᵏ是组合数，表示从n个元素中取k个的组合数：Cₙᵏ = n!/(k!(n-k)!)
示例
求y = x²eˣ的三阶导数
令u = x², v = eˣ
u' = 2x, u'' = 2, u''' = 0
v' = v'' = v''' = eˣ
由Leibniz公式：
y''' = C₃⁰u'''v + C₃¹u''v' + C₃²u'v'' + C₃³uv'''
= 0 + 3·2·eˣ + 3·2x·eˣ + 1·x²·eˣ
= (x² + 6x + 6)eˣ

▶ 微分运算
概述
微分运算研究函数微分的计算方法和性质，包括微分运算法则、一阶微分的形式不变性、隐函数求导法以及微分的应用等内容。
▶ 微分运算法则
1. d(u ± v) = du ± dv
2. d(uv) = u dv + v du
3. d(u/v) = (v du - u dv) / v² (v ≠ 0)
4. d(C) = 0 (C为常数)
▶ 一阶微分的形式不变性
定理表述
无论u是自变量还是中间变量，函数y = f(u)的微分形式都是dy = f'(u)du，这一性质称为一阶微分的形式不变性。
若y = f(u), u = g(x)，则
dy = f'(u)du = f'(u)g'(x)dx
▶ 隐函数求导法
方法介绍
对于由方程F(x, y) = 0确定的隐函数y = y(x)，求导时可以对方程两边同时微分，利用一阶微分的形式不变性，然后解出dy/dx。
示例
求由方程x² + y² = r²确定的隐函数y = y(x)的导数
对方程两边微分：d(x² + y²) = d(r²)
即2x dx + 2y dy = 0
解得：dy/dx = -x/y
▶ 微分的应用
误差估计
利用微分可以估计函数值的绝对误差和相对误差。若自变量x的绝对误差为Δx，则函数y = f(x)的绝对误差约为|dy| = |f'(x)|·|Δx|，相对误差约为|dy/y| = |f'(x)/f(x)|·|Δx|。
近似计算
当Δx很小时，利用微分可以近似计算函数的增量或函数值：f(x₀ + Δx) ≈ f(x₀) + f'(x₀)Δx。
示例：计算√4.01的近似值
令f(x) = √x, x₀ = 4, Δx = 0.01
f'(x) = 1/(2√x), f'(4) = 1/4
√4.01 = f(4 + 0.01) ≈ f(4) + f'(4)·0.01 = 2 + (1/4)·0.01 = 2.0025

▶ 微分中值定理
▶ Fermat定理
定理表述
Fermat定理是微分学中的一个基本定理，它指出如果函数在某点取得极值且在该点可导，则函数在该点的导数为零。
若函数f(x)在点x₀处取得极值，且f(x)在x₀处可导，则f'(x₀) = 0
几何意义
函数在极值点处的切线是水平的。
▶ Rolle定理
定理表述
若f(x)在[a,b]连续，在(a,b)可导，且f(a)=f(b)，则∃ξ∈(a,b)，使得f'(ξ)=0
▶ Lagrange中值定理
定理表述
若f(x)在[a,b]连续，在(a,b)可导，则∃ξ∈(a,b)，使得f'(ξ) = [f(b)-f(a)]/(b-a)
▶ Cauchy中值定理
定理表述
若f(x)和g(x)在[a,b]连续，在(a,b)可导，且g'(x)≠0，则∃ξ∈(a,b)，使得[f(b)-f(a)]/[g(b)-g(a)] = f'(ξ)/g'(ξ)

▶ L’Hospital法则
概述
L’Hospital法则是一种用于计算不定式极限的方法，适用于0/0型和∞/∞型不定式，以及可以转化为这两种类型的其他不定式。
▶ 基本形式（0/0型）
定理表述
若：
1. limₓ→a f(x) = 0, limₓ→a g(x) = 0
2. f(x)和g(x)在a的某去心邻域内可导，且g'(x) ≠ 0
3. limₓ→a f'(x)/g'(x) = A（或∞）
则limₓ→a f(x)/g(x) = limₓ→a f'(x)/g'(x) = A（或∞）
▶ 基本形式（∞/∞型）
定理表述
若：
1. limₓ→a f(x) = ∞, limₓ→a g(x) = ∞
2. f(x)和g(x)在a的某去心邻域内可导，且g'(x) ≠ 0
3. limₓ→a f'(x)/g'(x) = A（或∞）
则limₓ→a f(x)/g(x) = limₓ→a f'(x)/g'(x) = A（或∞）
▶ 其他不定型
0·∞型
对于0·∞型不定式，可以通过变形转化为0/0型或∞/∞型，再应用L’Hospital法则。
示例：limₓ→0⁺ x·lnx = limₓ→0⁺ lnx / (1/x) （转化为∞/∞型）
∞ - ∞型
对于∞ - ∞型不定式，通常通过通分或有理化转化为0/0型或∞/∞型。
示例：limₓ→1 [1/(x-1) - 1/lnx] = limₓ→1 [lnx - (x-1)] / [(x-1)lnx] （通分后转化为0/0型）
1^∞型
对于1^∞型不定式，可以利用对数恒等式y = e^lnx转化为0·∞型，再进一步转化为0/0型或∞/∞型。
示例：limₓ→0⁺ (1+x)^(1/x) = e^limₓ→0⁺ [ln(1+x)/x] （转化为0/0型）
0^0型和∞^0型
对于0^0型和∞^0型不定式，同样可以利用对数恒等式转化为0·∞型，再进一步转化为0/0型或∞/∞型。
示例：limₓ→0⁺ x^x = e^limₓ→0⁺ [xlnx] （转化为0·∞型）
▶ 注意事项
1. L’Hospital法则仅适用于0/0型和∞/∞型不定式，其他类型需要先转化
2. 每次应用L’Hospital法则前，需要检查是否满足条件
3. 若导数比值的极限不存在且不为∞，则L’Hospital法则失效，需用其他方法
4. 对于数列极限，可以先将其转化为函数极限，再应用L’Hospital法则

公式

▶ Lagrange余项
公式
f(x) = f(a) + f'(a)(x-a) + ... + f⁽ⁿ⁾(a)/n!·(x-a)ⁿ + f⁽ⁿ⁺¹⁾(ξ)/(n+1)!·(x-a)ⁿ⁺¹, ξ在a与x之间
▶ Maclaurin公式
公式
f(x) = f(0) + f'(0)x + f''(0)/2!·x² + ... + f⁽ⁿ⁾(0)/n!·xⁿ + o(xⁿ) (x→0)

▶ 点调性与单调区间
判定
在区间I内，若f'(x) > 0，则f在I上单调递增；若f'(x) < 0，则f在I上单调递减
▶ 极值与最大最小值
判定
若f'(x₀) = 0，且f''(x₀) > 0，则x₀为极小值点；若f''(x₀) < 0，则x₀为极大值点
全局最大/最小值需在闭区间上比较临界点与端点的函数值
▶ 凸性与凹性
判定
f为凸函数 ⇨ ∀λ∈[0,1], f(λx+(1-λ)y) ≤ λf(x)+(1-λ)f(y)
若f在区间内二次可导且f''(x) ≥ 0，则f凸；若f''(x) ≤ 0，则f凹
▶ 拐点
拐点是函数由凸变凹或由凹变凸的位置，常以f''(x)符号变化来判定。

▶ 定积分的定义（Riemann和）
定义
设f(x)在[a,b]上有界，将区间[a,b]分成n个小区间，第i个小区间的长度为Δx_i，在每个小区间上任取一点ξ_i，作和S = Σf(ξ_i)Δx_i。若当λ→0（λ为最大的小区间长度）时，S的极限存在，则称f(x)在[a,b]上可积，并称此极限为f(x)在[a,b]上的定积分，记为∫ₐᵇf(x)dx
可积条件
1. 若f(x)在[a,b]上连续，则f(x)在[a,b]上可积
2. 若f(x)在[a,b]上有界且只有有限个间断点，则f(x)在[a,b]上可积
