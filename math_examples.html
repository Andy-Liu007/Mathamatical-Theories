数列极限的定义例题

例题1：用ε-N定义证明limₙ→∞ (3n+1)/(2n-1) = 3/2
证明：∀ε > 0，∃N ∈ ℕ，当n > N时，|(3n+1)/(2n-1) - 3/2| < ε

解答：

计算|(3n+1)/(2n-1) - 3/2| = |5/(4n-2)|
对于任意ε > 0，取N = [ (5/ε + 2)/4 ] + 1
当n > N时，|(3n+1)/(2n-1) - 3/2| = 5/(4n-2) < ε
例题2：用ε-N定义证明limₙ→∞ 1/n² = 0
证明：∀ε > 0，∃N ∈ ℕ，当n > N时，|1/n² - 0| < ε

解答：

|1/n² - 0| = 1/n² < ε ⇒ n > √(1/ε)
取N = [√(1/ε)] + 1，当n > N时，1/n² < ε

函数极限的唯一性例题

例题1：验证函数f(x) = (x² - 4)/(x - 2)在x→2时的极限唯一性

解答：

当x ≠ 2时，f(x) = x + 2
limₓ→2 f(x) = limₓ→2 (x + 2) = 4
由函数极限的唯一性定理，该极限唯一
