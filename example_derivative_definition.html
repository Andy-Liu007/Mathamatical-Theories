例题1：用导数定义求函数的导数

用导数的定义求函数f(x) = x²在x = 1处的导数。

解答：

根据导数的定义：f'(x₀) = limₕ→₀ [f(x₀+h) - f(x₀)] / h

代入x₀ = 1，f(x) = x²：
f'(1) = limₕ→₀ [(1+h)² - 1²] / h
= limₕ→₀ [1 + 2h + h² - 1] / h
= limₕ→₀ [2h + h²] / h
= limₕ→₀ [2 + h]
= 2

因此，f'(1) = 2

例题2：求函数的微分

求函数y = x³ + 2x² - 5x + 1的微分。

解答：

首先求函数的导数：y' = 3x² + 4x - 5

根据微分的定义：dy = y' dx

代入导数结果：dy = (3x² + 4x - 5) dx

因此，函数的微分为dy = (3x² + 4x - 5) dx

例题3：用微分进行近似计算

利用微分近似计算√4.01的值。

解答：

设f(x) = √x，取x₀ = 4，Δx = 0.01

首先求f(x)的导数：f'(x) = 1/(2√x)

计算f(x₀)和f'(x₀)：
f(4) = √4 = 2
f'(4) = 1/(2√4) = 1/4

根据微分近似公式：f(x₀+Δx) ≈ f(x₀) + f'(x₀)Δx

代入数值：√4.01 = f(4+0.01) ≈ f(4) + f'(4)×0.01 = 2 + (1/4)×0.01 = 2 + 0.0025 = 2.0025

因此，√4.01 ≈ 2.0025（精确值约为2.002498439）
