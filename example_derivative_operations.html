例题1：基本导数公式的应用

求下列函数的导数：

  1. f(x) = x⁵
  2. g(x) = sinx
  3. h(x) = eˣ
  4. k(x) = ln|x|

解答：

1. 对于f(x) = x⁵，使用幂函数导数公式：(xⁿ)' = nxⁿ⁻¹
f'(x) = 5x⁴

2. 对于g(x) = sinx，使用三角函数导数公式：(sinx)' = cosx
g'(x) = cosx

3. 对于h(x) = eˣ，使用指数函数导数公式：(eˣ)' = eˣ
h'(x) = eˣ

4. 对于k(x) = ln|x|，使用对数函数导数公式：(ln|x|)' = 1/x
k'(x) = 1/x

例题2：导数的四则运算

求函数f(x) = x²sinx + 3eˣ的导数。

解答：

首先，我们可以将函数拆分为两部分：f(x) = u(x) + v(x)，其中u(x) = x²sinx，v(x) = 3eˣ

对于u(x) = x²sinx，使用乘积法则：(uv)' = u'v + uv'
u'(x) = (x²)'sinx + x²(sinx)' = 2x sinx + x² cosx

对于v(x) = 3eˣ，使用常数乘法法则和指数函数导数公式：
v'(x) = 3(eˣ)' = 3eˣ

因此，f'(x) = u'(x) + v'(x) = 2x sinx + x² cosx + 3eˣ

例题3：复合函数求导

求函数y = sin(x² + 1)的导数。

解答：

设u = x² + 1，则y = sinu

使用链式法则：dy/dx = dy/du · du/dx

计算dy/du：dy/du = (sinu)' = cosu

计算du/dx：du/dx = (x² + 1)' = 2x

因此，dy/dx = cosu · 2x = 2x cos(x² + 1)

例题4：反函数求导法则（反正弦函数）

已知y = arcsinx，求dy/dx。

解答：

因为y = arcsinx，所以x = siny，其中y ∈ [-π/2, π/2]

对x = siny两边关于x求导：1 = cosy · dy/dx

因此，dy/dx = 1/cosy

由于y ∈ [-π/2, π/2]，所以cosy ≥ 0，因此cosy = √(1 - sin²y) = √(1 - x²)

所以，dy/dx = 1/√(1 - x²)

例题5：反函数求导法则（反正切函数）

已知y = arctanx，求dy/dx。

解答：

因为y = arctanx，所以x = tany，其中y ∈ (-π/2, π/2)

对x = tany两边关于x求导：1 = sec²y · dy/dx

因此，dy/dx = 1/sec²y

利用三角恒等式：sec²y = 1 + tan²y，所以sec²y = 1 + x²

因此，dy/dx = 1/(1 + x²)

所以，dy/dx = 1/(1 + x²)

例题6：对数求导法

求函数y = xˣ的导数（x > 0）。

解答：

这是一个幂指函数，使用对数求导法：

1. 两边取自然对数：ln|y| = ln|xˣ| = x ln|x|（因为x > 0，所以ln|x| = lnx）

2. 两边对x求导：(1/y)·y' = lnx + x·(1/x) = lnx + 1

3. 解出y'：y' = y·(lnx + 1) = xˣ·(lnx + 1)
