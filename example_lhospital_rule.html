例题1：0/0型不定式

计算极限：limₓ→0 (sinx) / x

解答：

当x→0时，sinx→0，x→0，因此这是0/0型不定式

应用L’Hospital法则，对分子和分母分别求导：
limₓ→0 (sinx) / x = limₓ→0 (cosx) / 1 = cos0 / 1 = 1

因此，limₓ→0 (sinx) / x = 1

例题2：∞/∞型不定式

计算极限：limₓ→+∞ (lnx) / x

解答：

当x→+∞时，lnx→+∞，x→+∞，因此这是∞/∞型不定式

应用L’Hospital法则，对分子和分母分别求导：
limₓ→+∞ (lnx) / x = limₓ→+∞ (1/x) / 1 = limₓ→+∞ 1/x = 0

因此，limₓ→+∞ (lnx) / x = 0

例题3：0·∞型不定式

计算极限：limₓ→0⁺ x · lnx

解答：

当x→0⁺时，x→0，lnx→-∞，因此这是0·∞型不定式

将其转化为∞/∞型不定式：limₓ→0⁺ lnx / (1/x)

应用L’Hospital法则，对分子和分母分别求导：
limₓ→0⁺ (1/x) / (-1/x²) = limₓ→0⁺ (-x) = 0

因此，limₓ→0⁺ x · lnx = 0

例题4：∞ - ∞型不定式

计算极限：limₓ→1 [1/(x-1) - 1/lnx]

解答：

当x→1时，1/(x-1)→±∞，1/lnx→±∞，因此这是∞ - ∞型不定式

将其通分转化为0/0型不定式：
limₓ→1 [lnx - (x-1)] / [(x-1)lnx]

应用L’Hospital法则，对分子和分母分别求导：
分子导数：(1/x) - 1 = (1 - x)/x
分母导数：lnx + (x-1)(1/x) = lnx + 1 - 1/x

得到：limₓ→1 [(1 - x)/x] / [lnx + 1 - 1/x]

这仍然是0/0型不定式，再次应用L’Hospital法则：
分子导数：(-1·x - (1 - x)·1) / x² = (-x - 1 + x) / x² = -1/x²
分母导数：(1/x) + (1/x²) = (x + 1)/x²

得到：limₓ→1 (-1/x²) / [(x + 1)/x²] = limₓ→1 (-1) / (x + 1) = -1/2

因此，limₓ→1 [1/(x-1) - 1/lnx] = -1/2

例题5：1^∞型不定式

计算极限：limₓ→0⁺ (1+x)^(1/x)

解答：

当x→0⁺时，(1+x)→1，1/x→+∞，因此这是1^∞型不定式

利用对数恒等式：y = e^lnx，令y = (1+x)^(1/x)，则lny = (1/x)ln(1+x)

计算limₓ→0⁺ lny = limₓ→0⁺ [ln(1+x)] / x，这是0/0型不定式

应用L’Hospital法则：limₓ→0⁺ [1/(1+x)] / 1 = 1/(1+0) = 1

因此，limₓ→0⁺ y = e^limₓ→0⁺ lny = e^1 = e

所以，limₓ→0⁺ (1+x)^(1/x) = e

例题6：0^0型不定式

计算极限：limₓ→0⁺ x^x

解答：

当x→0⁺时，x→0，x→0，因此这是0^0型不定式

利用对数恒等式：y = e^lnx，令y = x^x，则lny = x ln x

计算limₓ→0⁺ lny = limₓ→0⁺ x ln x，这是0·∞型不定式，已在例题3中求得其值为0

因此，limₓ→0⁺ y = e^limₓ→0⁺ lny = e^0 = 1

所以，limₓ→0⁺ x^x = 1

例题7：∞^0型不定式

计算极限：limₓ→+∞ x^(1/x)

解答：

当x→+∞时，x→+∞，1/x→0，因此这是∞^0型不定式

利用对数恒等式：y = e^lnx，令y = x^(1/x)，则lny = (1/x) ln x

计算limₓ→+∞ lny = limₓ→+∞ [lnx] / x，这是∞/∞型不定式，已在例题2中求得其值为0

因此，limₓ→+∞ y = e^limₓ→+∞ lny = e^0 = 1

所以，limₓ→+∞ x^(1/x) = 1
