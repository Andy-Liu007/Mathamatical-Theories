例题1：局部有界性定理的验证

验证函数f(x) = x²sin(1/x)在x→0时的局部有界性。

解答：

首先计算x→0时的极限：limₓ→0 x²sin(1/x) = 0（利用夹逼准则，|x²sin(1/x)| ≤ x²）

根据函数极限的局部有界性定理：若limₓ→a f(x) = A，则∃δ > 0 和 M > 0，使得当0 < |x - a| < δ时，|f(x)| ≤ M

这里a = 0，A = 0，因此存在δ > 0和M > 0，当0 < |x| < δ时，|x²sin(1/x)| ≤ M

具体来说，取δ = 1，M = 1²·1 = 1，则当0 < |x| < 1时，|x²sin(1/x)| ≤ |x²|·1 ≤ 1² = 1 = M

因此，函数f(x)在x→0时局部有界

例题2：局部有界性的应用

已知limₓ→1 f(x) = 5，证明存在δ > 0，使得当0 < |x - 1| < δ时，|f(x)| < 10。

解答：

根据函数极限的局部有界性定理：若limₓ→a f(x) = A，则∃δ > 0 和 M > 0，使得当0 < |x - a| < δ时，|f(x)| ≤ M

这里a = 1，A = 5，因此存在δ > 0和M > 0，当0 < |x - 1| < δ时，|f(x)| ≤ M

具体来说，取ε = 5，由极限定义，∃δ₁ > 0，当0 < |x - 1| < δ₁时，|f(x) - 5| < 5
则|f(x)| = |(f(x) - 5) + 5| ≤ |f(x) - 5| + 5 < 5 + 5 = 10

取δ = δ₁，则当0 < |x - 1| < δ时，|f(x)| < 10，得证
