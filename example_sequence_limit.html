例题1：用ε-N定义证明数列极限

使用ε-N定义证明：limₙ→∞ (2n+1)/(3n-2) = 2/3

解答：

首先计算|(2n+1)/(3n-2) - 2/3|：
|(2n+1)/(3n-2) - 2/3| = |[3(2n+1) - 2(3n-2)]/[3(3n-2)]| = |(6n+3 - 6n+4)/(9n-6)| = |7/(9n-6)|

对于任意给定的ε > 0，我们需要找到N ∈ ℕ，使得当n > N时：
|7/(9n-6)| < ε

化简不等式：7/(9n-6) < ε ⇒ 9n - 6 > 7/ε ⇒ n > (7/ε + 6)/9

取N = [ (7/ε + 6)/9 ] + 1（其中[·]表示取整函数），则当n > N时：
|(2n+1)/(3n-2) - 2/3| = 7/(9n-6) < 7/(9N-6) < ε

因此，根据ε-N定义，limₙ→∞ (2n+1)/(3n-2) = 2/3

例题2：数列极限的四则运算

已知limₙ→∞ aₙ = 3，limₙ→∞ bₙ = 5，求limₙ→∞ (2aₙ - 3bₙ + 7)

解答：

根据数列极限的四则运算法则：
limₙ→∞ (2aₙ - 3bₙ + 7) = 2·limₙ→∞ aₙ - 3·limₙ→∞ bₙ + limₙ→∞ 7

代入已知极限值：
= 2·3 - 3·5 + 7
= 6 - 15 + 7
= -2

因此，limₙ→∞ (2aₙ - 3bₙ + 7) = -2

例题3：夹逼准则的应用

求limₙ→∞ (n!)/(nⁿ)

解答：

首先注意到n! = 1·2·3·...·n，而nⁿ = n·n·n·...·n

因此，当n ≥ 2时，n! = 1·2·3·...·n ≤ 1·n·n·...·n = nⁿ⁻¹
且n! ≥ 1·2·1·...·1 = 2（当n ≥ 2时）

但更精确的估计是：
当n ≥ 2时，n! = 1·2·3·...·n ≤ 1·n·n·...·n = nⁿ⁻¹
且n! ≥ 1·n·1·...·1 = n

因此，我们有：n ≤ n! ≤ nⁿ⁻¹
两边除以nⁿ：1/nⁿ⁻¹ ≤ n!/nⁿ ≤ 1/n

由于limₙ→∞ 1/nⁿ⁻¹ = 0，limₙ→∞ 1/n = 0
根据夹逼准则，limₙ→∞ (n!)/nⁿ = 0

例题4：Cauchy收敛准则的应用

证明数列aₙ = 1 + 1/2 + 1/3 + ... + 1/n发散（调和级数）

解答：

使用Cauchy收敛准则的否定形式：数列{aₙ}发散 ⇨ ∃ε₀ > 0，∀N ∈ ℕ，∃m, n > N，使得|aₘ - aₙ| ≥ ε₀

取ε₀ = 1/2，对于任意N ∈ ℕ，取n = N + 1，m = 2N + 2
则|aₘ - aₙ| = |(1 + 1/2 + ... + 1/(2N+2)) - (1 + 1/2 + ... + 1/(N+1))| = |1/(N+2) + 1/(N+3) + ... + 1/(2N+2)|

注意到共有N+1项，每项都 ≥ 1/(2N+2)，因此：
|aₘ - aₙ| ≥ (N+1)·1/(2N+2) = (N+1)/(2(N+1)) = 1/2 = ε₀

因此，根据Cauchy收敛准则的否定形式，调和级数发散
