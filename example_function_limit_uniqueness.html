例题1：函数极限唯一性的验证

验证函数f(x) = (x² - 4)/(x - 2)在x→2时的极限是否唯一。

解答：

首先化简函数：当x ≠ 2时，(x² - 4)/(x - 2) = (x - 2)(x + 2)/(x - 2) = x + 2

计算x→2时的极限：limₓ→2 f(x) = limₓ→2 (x + 2) = 4

假设存在另一个极限B ≠ 4，则根据函数极限的唯一性定理，这是不可能的

因此，函数f(x)在x→2时的极限唯一，为4

例题2：唯一性定理的应用

已知limₓ→0 f(x) = L，且f(x) = (sinx)/x + cx²，求常数L和c的值。

解答：

首先，我们知道limₓ→0 (sinx)/x = 1（重要极限）

根据函数极限的四则运算法则：
limₓ→0 f(x) = limₓ→0 [(sinx)/x + cx²] = limₓ→0 (sinx)/x + c·limₓ→0 x² = 1 + c·0 = 1

由函数极限的唯一性定理，limₓ→0 f(x) = L = 1，因此L = 1

由于极限值与c无关，c可以是任意实数

结论：L = 1，c为任意实数
