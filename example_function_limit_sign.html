例题1：局部保号性定理的验证

验证函数f(x) = x³ + 2x在x→1时的局部保号性。

解答：

首先计算x→1时的极限：limₓ→1 f(x) = limₓ→1 (x³ + 2x) = 1³ + 2·1 = 3 > 0

根据函数极限的局部保号性定理：若limₓ→a f(x) = A > 0，则∃δ > 0，使得当0 < |x - a| < δ时，f(x) > 0

这里a = 1，A = 3 > 0，因此存在δ > 0，当0 < |x - 1| < δ时，f(x) > 0

具体来说，取ε = 1，由极限定义，∃δ₁ > 0，当0 < |x - 1| < δ₁时，|f(x) - 3| < 1
则f(x) > 3 - 1 = 2 > 0

因此，函数f(x)在x→1时局部保号，即存在δ > 0，当0 < |x - 1| < δ时，f(x) > 0

例题2：局部保号性的应用

已知limₓ→2 f(x) = 4 > 0，证明存在δ > 0，使得当0 < |x - 2| < δ时，f(x) > 2。

解答：

根据函数极限的局部保号性定理：若limₓ→a f(x) = A > 0，则∃δ > 0，使得当0 < |x - a| < δ时，f(x) > 0

这里a = 2，A = 4 > 0，我们需要证明存在δ > 0，使得当0 < |x - 2| < δ时，f(x) > 2

取ε = 2，由极限定义，∃δ₁ > 0，当0 < |x - 2| < δ₁时，|f(x) - 4| < 2
则f(x) > 4 - 2 = 2

取δ = δ₁，则当0 < |x - 2| < δ时，f(x) > 2，得证

这说明局部保号性定理不仅保证函数值保持符号，还可以保证函数值与极限值足够接近时，保持一定的大小关系
