例题1：Fermat定理的应用

证明多项式P(x) = x³ - 3x + 1在区间[-2, 2]内至少有一个临界点。

解答：

首先，多项式P(x) = x³ - 3x + 1在实数域上连续且可导

计算P(x)的导数：P'(x) = 3x² - 3 = 3(x² - 1) = 3(x - 1)(x + 1)

令P'(x) = 0，解得x = 1和x = -1，这两个点都在区间[-2, 2]内

根据Fermat定理，这两个点都是P(x)的临界点

因此，P(x)在区间[-2, 2]内至少有一个临界点（实际上有两个）

例题2：Rolle定理的应用

验证Rolle定理对函数f(x) = x² - 4x + 3在区间[1, 3]上的适用性，并求出满足定理条件的ξ值。

解答：

首先检查Rolle定理的三个条件：
1. f(x)在[1, 3]上连续：f(x)是多项式，在实数域上连续，因此在[1, 3]上连续
2. f(x)在(1, 3)内可导：f(x)是多项式，在实数域上可导，因此在(1, 3)内可导
3. f(1) = f(3)：计算得f(1) = 1 - 4 + 3 = 0，f(3) = 9 - 12 + 3 = 0，因此f(1) = f(3)

三个条件都满足，因此存在ξ∈(1, 3)，使得f'(ξ) = 0

计算f(x)的导数：f'(x) = 2x - 4

令f'(ξ) = 0，解得2ξ - 4 = 0 ⇒ ξ = 2

验证ξ = 2∈(1, 3)，满足条件

因此，满足Rolle定理条件的ξ值为2

例题3：Lagrange中值定理的应用

应用Lagrange中值定理证明：对任意实数a和b，|sinb - sina| ≤ |b - a|。

解答：

设f(x) = sinx，考虑区间[a, b]（假设a < b，a > b的情况类似）

检查Lagrange中值定理的条件：
1. f(x) = sinx在[a, b]上连续
2. f(x) = sinx在(a, b)内可导，且f'(x) = cosx

根据Lagrange中值定理，存在ξ∈(a, b)，使得：
f(b) - f(a) = f'(ξ)(b - a)
即sinb - sina = cosξ(b - a)

取绝对值两边：|sinb - sina| = |cosξ||b - a|

由于|cosξ| ≤ 1对任意实数ξ成立，因此：
|sinb - sina| ≤ 1·|b - a| = |b - a|

当a = b时，等号成立

因此，对任意实数a和b，|sinb - sina| ≤ |b - a|

例题4：Cauchy中值定理的应用

设f(x) = x²，g(x) = x³，在区间[1, 2]上验证Cauchy中值定理，并求出满足定理条件的ξ值。

解答：

首先检查Cauchy中值定理的条件：
1. f(x)和g(x)在[1, 2]上连续：两者都是多项式，在实数域上连续
2. f(x)和g(x)在(1, 2)内可导：两者都是多项式，在实数域上可导
3. g'(x) = 3x² ≠ 0在(1, 2)内成立：因为x∈(1, 2)时，x² > 0，所以g'(x) > 0

根据Cauchy中值定理，存在ξ∈(1, 2)，使得：
[f(2) - f(1)] / [g(2) - g(1)] = f'(ξ) / g'(ξ)

计算左边：
[f(2) - f(1)] = 4 - 1 = 3
[g(2) - g(1)] = 8 - 1 = 7
因此左边为3/7

计算右边：
f'(ξ) = 2ξ，g'(ξ) = 3ξ²，因此右边为2ξ / 3ξ² = 2 / 3ξ

令左右两边相等：3/7 = 2 / 3ξ
解得：ξ = (2×7) / (3×3) = 14/9 ≈ 1.555

验证ξ = 14/9∈(1, 2)，满足条件

因此，满足Cauchy中值定理条件的ξ值为14/9
