例题1：sinx的Maclaurin展开

求函数f(x) = sinx的Maclaurin展开（到x⁵项）

解答：

Maclaurin展开公式：f(x) = f(0) + f'(0)x + f''(0)/2!·x² + f'''(0)/3!·x³ + f⁽⁴⁾(0)/4!·x⁴ + f⁽⁵⁾(0)/5!·x⁵ + o(x⁵)

计算各阶导数在x=0处的值：
f(x) = sinx，f(0) = 0
f'(x) = cosx，f'(0) = 1
f''(x) = -sinx，f''(0) = 0
f'''(x) = -cosx，f'''(0) = -1
f⁽⁴⁾(x) = sinx，f⁽⁴⁾(0) = 0
f⁽⁵⁾(x) = cosx，f⁽⁵⁾(0) = 1

代入Maclaurin公式：
sinx = 0 + 1·x + 0/2!·x² + (-1)/3!·x³ + 0/4!·x⁴ + 1/5!·x⁵ + o(x⁵)

化简得：
sinx = x - x³/6 + x⁵/120 + o(x⁵)

例题2：cosx的Maclaurin展开

求函数f(x) = cosx的Maclaurin展开（到x⁶项）

解答：

Maclaurin展开公式：f(x) = f(0) + f'(0)x + f''(0)/2!·x² + f'''(0)/3!·x³ + f⁽⁴⁾(0)/4!·x⁴ + f⁽⁵⁾(0)/5!·x⁵ + f⁽⁶⁾(0)/6!·x⁶ + o(x⁶)

计算各阶导数在x=0处的值：
f(x) = cosx，f(0) = 1
f'(x) = -sinx，f'(0) = 0
f''(x) = -cosx，f''(0) = -1
f'''(x) = sinx，f'''(0) = 0
f⁽⁴⁾(x) = cosx，f⁽⁴⁾(0) = 1
f⁽⁵⁾(x) = -sinx，f⁽⁵⁾(0) = 0
f⁽⁶⁾(x) = -cosx，f⁽⁶⁾(0) = -1

代入Maclaurin公式：
cosx = 1 + 0·x + (-1)/2!·x² + 0/3!·x³ + 1/4!·x⁴ + 0/5!·x⁵ + (-1)/6!·x⁶ + o(x⁶)

化简得：
cosx = 1 - x²/2 + x⁴/24 - x⁶/720 + o(x⁶)

例题3：eˣ的Maclaurin展开

求函数f(x) = eˣ的Maclaurin展开（到x⁴项）

解答：

Maclaurin展开公式：f(x) = f(0) + f'(0)x + f''(0)/2!·x² + f'''(0)/3!·x³ + f⁽⁴⁾(0)/4!·x⁴ + o(x⁴)

由于eˣ的所有阶导数都是eˣ，因此：
f⁽ⁿ⁾(x) = eˣ，f⁽ⁿ⁾(0) = e⁰ = 1（对所有n）

代入Maclaurin公式：
eˣ = 1 + 1·x + 1/2!·x² + 1/3!·x³ + 1/4!·x⁴ + o(x⁴)

化简得：
eˣ = 1 + x + x²/2 + x³/6 + x⁴/24 + o(x⁴)

例题4：ln(1+x)的Maclaurin展开

求函数f(x) = ln(1+x)的Maclaurin展开（到x⁴项）

解答：

Maclaurin展开公式：f(x) = f(0) + f'(0)x + f''(0)/2!·x² + f'''(0)/3!·x³ + f⁽⁴⁾(0)/4!·x⁴ + o(x⁴)

计算各阶导数在x=0处的值：
f(x) = ln(1+x)，f(0) = 0
f'(x) = 1/(1+x)，f'(0) = 1
f''(x) = -1/(1+x)²，f''(0) = -1
f'''(x) = 2/(1+x)³，f'''(0) = 2
f⁽⁴⁾(x) = -6/(1+x)⁴，f⁽⁴⁾(0) = -6

代入Maclaurin公式：
ln(1+x) = 0 + 1·x + (-1)/2!·x² + 2/3!·x³ + (-6)/4!·x⁴ + o(x⁴)

化简得：
ln(1+x) = x - x²/2 + x³/3 - x⁴/4 + o(x⁴)

例题5：(1+x)^α的Maclaurin展开

求函数f(x) = (1+x)^α的Maclaurin展开（到x³项），其中α为常数

解答：

Maclaurin展开公式：f(x) = f(0) + f'(0)x + f''(0)/2!·x² + f'''(0)/3!·x³ + o(x³)

计算各阶导数在x=0处的值：
f(x) = (1+x)^α，f(0) = 1
f'(x) = α(1+x)^(α-1)，f'(0) = α
f''(x) = α(α-1)(1+x)^(α-2)，f''(0) = α(α-1)
f'''(x) = α(α-1)(α-2)(1+x)^(α-3)，f'''(0) = α(α-1)(α-2)

代入Maclaurin公式：
(1+x)^α = 1 + αx + α(α-1)/2!·x² + α(α-1)(α-2)/3!·x³ + o(x³)

化简得：
(1+x)^α = 1 + αx + α(α-1)x²/2 + α(α-1)(α-2)x³/6 + o(x³)
