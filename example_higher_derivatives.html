例题1：求函数的高阶导数

求函数f(x) = x³ + 2x² - 5x + 1的一阶、二阶和三阶导数。

解答：

首先求一阶导数：
f'(x) = 3x² + 4x - 5

然后求二阶导数：
f''(x) = (f'(x))' = (3x² + 4x - 5)' = 6x + 4

最后求三阶导数：
f'''(x) = (f''(x))' = (6x + 4)' = 6

因此，f'(x) = 3x² + 4x - 5，f''(x) = 6x + 4，f'''(x) = 6

例题2：常见函数的高阶导数

求下列函数的n阶导数：

  1. f(x) = eˣ
  2. g(x) = sinx
  3. h(x) = cosx
  4. k(x) = lnx

解答：

1. 对于f(x) = eˣ，其导数保持不变：
f⁽ⁿ⁾(x) = eˣ

2. 对于g(x) = sinx，其n阶导数具有周期性：
g⁽ⁿ⁾(x) = sin(x + nπ/2)

3. 对于h(x) = cosx，其n阶导数也具有周期性：
h⁽ⁿ⁾(x) = cos(x + nπ/2)

4. 对于k(x) = lnx，其n阶导数为：
k⁽ⁿ⁾(x) = (-1)ⁿ⁻¹ (n-1)! / xⁿ

例题3：Leibniz公式的应用

使用Leibniz公式求函数y = x²eˣ的三阶导数。

解答：

Leibniz公式：(uv)⁽ⁿ⁾ = Σₖ=₀ⁿ Cₙᵏ u⁽ⁿ⁻ᵏ⁾ v⁽ᵏ⁾，其中Cₙᵏ是组合数

令u = x²，v = eˣ，则需要计算u的各阶导数和v的各阶导数：

u的各阶导数：
u = x²
u' = 2x
u'' = 2
u''' = 0
u⁽ᵏ⁾ = 0 (k ≥ 4)

v = eˣ的各阶导数：
v⁽ᵏ⁾ = eˣ (对任意k ≥ 0)

使用Leibniz公式计算三阶导数：
y''' = (x²eˣ)''' = Σₖ=₀³ C₃ᵏ (x²)⁽³⁻ᵏ⁾ (eˣ)⁽ᵏ⁾

展开求和：
y''' = C₃⁰ (x²)''' eˣ + C₃¹ (x²)'' (eˣ)' + C₃² (x²)' (eˣ)'' + C₃³ (x²) (eˣ)'''

计算各组合数：
C₃⁰ = 1, C₃¹ = 3, C₃² = 3, C₃³ = 1

代入各阶导数：
y''' = 1·0·eˣ + 3·2·eˣ + 3·2x·eˣ + 1·x²·eˣ
y''' = 0 + 6eˣ + 6xeˣ + x²eˣ
y''' = eˣ(x² + 6x + 6)

因此，y''' = eˣ(x² + 6x + 6)

例题4：Leibniz公式的进一步应用

求函数y = xsinx的四阶导数。

解答：

使用Leibniz公式：(uv)⁽ⁿ⁾ = Σₖ=₀ⁿ Cₙᵏ u⁽ⁿ⁻ᵏ⁾ v⁽ᵏ⁾

令u = x，v = sinx，则：

u的各阶导数：
u = x
u' = 1
u'' = 0
u''' = 0
u⁽⁴⁾ = 0

v = sinx的各阶导数：
v = sinx
v' = cosx = sin(x + π/2)
v'' = -sinx = sin(x + π)
v''' = -cosx = sin(x + 3π/2)
v⁽⁴⁾ = sinx = sin(x + 2π)

使用Leibniz公式计算四阶导数：
y⁽⁴⁾ = (xsinx)⁽⁴⁾ = Σₖ=₀⁴ C₄ᵏ (x)⁽⁴⁻ᵏ⁾ (sinx)⁽ᵏ⁾

展开求和：
y⁽⁴⁾ = C₄⁰ (x)⁽⁴⁾ sinx + C₄¹ (x)⁽³⁾ (sinx)' + C₄² (x)'' (sinx)'' + C₄³ (x)' (sinx)''' + C₄⁴ (x) (sinx)⁽⁴⁾

计算各组合数：
C₄⁰ = 1, C₄¹ = 4, C₄² = 6, C₄³ = 4, C₄⁴ = 1

代入各阶导数：
y⁽⁴⁾ = 1·0·sinx + 4·0·cosx + 6·0·(-sinx) + 4·1·(-cosx) + 1·x·sinx
y⁽⁴⁾ = 0 + 0 + 0 - 4cosx + xsinx
y⁽⁴⁾ = xsinx - 4cosx

因此，y⁽⁴⁾ = xsinx - 4cosx
